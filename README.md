# FASTFOOD README

## Cách chạy

- Web (thư mục `web`):

  ```powershell
  cd web
  npm install (lần đầu clone)
  npm run dev
  ```

- Mobile (thư mục `mobile`):

  ```powershell
  cd mobile
  npm install (lần đầu clone)
  npm start
  ```
